# Notes-App

For Adding In Notes
use the command 
For Example
node app.js add --title="Hello" --body="World"  

For Removing In Notes
use the command
For Example
node app.js remove --title="Hello"

For Listing In the Notes
use the command 
For Example
node app.js list

For Reading in the Notes
use the command
node app.js read --title="Hello"
